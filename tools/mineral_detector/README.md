# 矿物探测仪

性能较高且耐用的小型探测仪，可以为你寻找周围矿物

利用了声波雷达可视化技术，为你显示周围矿物

## 制作配方

> 使用工作台合成

<img src="./tools/mineral_detector/craft.jpg">

### 材料总需

* 铁 x3
* 音符盒 x1
* 红石 x1
* 回响碎片 x1
* 铜 x1

## 特性

手持矿物探测仪右键可以高亮显示 以你为中心的半径5格方块内 所有矿物，且其他人也可见
