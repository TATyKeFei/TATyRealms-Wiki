# 镘刀

现实生活中常见于建筑工人用来为盖房子的砖涂抹水泥的工具

## 制作配方

<img src="./tools/trowel/img/craft.jpg">

### 材料总需

* 铁锭 x1
* 木棍 x1

## 使用方法

将方块放在快捷栏时，使用镘刀右键地面可随机放置快捷栏里的方块

<img src="./tools/trowel/img/demo.gif">

> [!TIP]
> 快捷栏是指玩家键盘上可用数字键或鼠标滚轮快捷切换物品的物品槽位
>
> 如下图中 0~8 的位置
>
> ![](./img/Items_slot_number.png)
