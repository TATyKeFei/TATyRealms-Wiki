## Q群信息转发

服务器消息与QQ群消息是相互转发的，q群消息在游戏里能看到，q群也同样可以看到游戏内消息

如果玩家客户端有 ChatImage 模组可以在游戏内查看q群里发的图片消息

## 表情符号列表

你也可以使用以下解析符号发送表情

> 由于markdown的原因会被识别成emoji表情，这里打了中文符号
>
> 请自己将中文符号换成英文符号

| 符号        | 描述        |
| :---        | :---       |
| ：angry：   | 生气        |
| ：cry：     | 泪目        |
| ：dead：    | 死掉了      |
| ：fear：    | 害怕        |
| ：frown：   | 皱眉        |
| ：hehe：    | 呵呵        |
| ：mmh：     | 啊?         |
| ：sleeping：| 睡觉了      |
| ：smile：   | 笑          |
| ：tcoin：   | TCoin       |
| ：think：   | 思考        |
| ：think1：  | 想不出来    |
| ：oh：      | 原来如此!!  |
| ：what：    | 啥东西      |
| ：wink：    | 眨眼~       |
