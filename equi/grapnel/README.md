# 抓钩

我们的距离，并不遥远，无论是心灵，还是身躯，即使跨越生死

我也会尽我所能，向你所在的方向，奋力前进，直至你的身边

## 合成配方

> 使用工作台合成

<img src="./equi/grapnel/img/craft.jpg">

### 材料总需

* 铁锭 x3
* 线 x1
* 弓 x1
* 橡木按钮 x1
* 红石粉 x1

## 使用教程

右键可以抛出一个钩子，等钩子抓到方块后再右键，它可以把你拉到你想去的地方

### 特性

* 抓钩使用次数限制都为 100次
* 如果把抓钩放在左手使用是不生效的
* 抛出钩子后，如果玩家在空中会获得漂浮效果
