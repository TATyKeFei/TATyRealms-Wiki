# 小唠嗑

## 你这雨伞是不是代码写反了? 雨伞怎么不是缓落而是升天?

原本是这样打算的，但考虑到服务器没有摔落伤害导致雨伞显得有些尴尬选择了升天

毕竟动漫里的人都能用雨伞起飞，mc里也能起飞不过分吧

## 雨伞盾牌

这是个废弃的草案，考虑到盾牌右键举盾和按住右键升天冲突

原本想换成狂按潜行键升天，但对我这种天天敲键盘的对小拇指不太友好废弃了
