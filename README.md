## 关于TATySMP

TATySMP由“[TATyKeFei](https://bbs.tatysmp.love/index.php?members/tatykefei.1/)”于2021年8月1日创建。最开始是基岩版服务器为朋友之间方便联机而开。

后来慢慢的发现了插件的神奇力量逐渐走向互通服这个不归路发展成了Java/基岩互通服

> 你知道吗? TATySMP服务器和网站只有腐竹一人维护，整个wiki内容都是TATyKeFei写的
