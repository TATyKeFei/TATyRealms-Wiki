## 指令

```
/sprays upload <图片url链接> --上传喷漆图片到服务器
示例: /sprays upload https://www.tatysmp.love/assets/images/TATyKeFei.png

/givesprays --获得喷漆灌
```

## 使用教程

> 小提示: java玩家可快速双击f来进行喷漆

在上传完图片加载完成后，使用喷漆罐"右键/点击"地面来进行喷漆。

每次喷漆会消耗100Tcoin

使用冷却40s时间
