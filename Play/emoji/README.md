## 表情符号列表

你也可以使用以下解析符号发送表情

> 由于markdown的原因会被识别成emoji表情，打了中文符号
>
> 自己将中文符号换成英文符号

| 符号        | 描述        |
| ----------- | ----------- |
| ：angry：   | 生气        |
| ：cry：     | 泪目        |
| ：dead：    | 死掉了      |
| ：fear：    | 害怕        |
| ：frown：   | 皱眉        |
| ：hehe：    | 呵呵        |
| ：mmh：     | 啊?         |
| ：sleeping：| 睡觉了      |
| ：smile：   | 笑          |
| ：tcoin：   | TCoin       |
| ：think：   | 思考        |
| ：think1：  | 想不出来    |
| ：oh：      | 原来如此!!  |
| ：what：    | 啥东西      |
| ：wink：    | 眨眼~       |
